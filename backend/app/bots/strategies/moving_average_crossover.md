The strategy works in pairs:                                                                                                                                                                                       
                                                                                                                                                                                                                                     
BUY  → you open a position  (you own BTC)                                                                                                                                                                                          
SELL → you close a position (you sell BTC back for USDT)                                                                                                                                                                           
                                                                                                                                                                                                                                 
The state is tracked by _is_position_open in the strategy:                                                                                                                                                                         
                                                                                                                                                                                                                                 
## On BUY signal:                                                                                                                                                                                                                   
if not self._is_position_open:   # only buy if we don't already hold BTC                                                                                                                                                           
  self._is_position_open = True                                                                                                                                                                                                  
  → emit BUY order                                                                                                                                                                                                               
                                                                                                                                                                                                                                 
## On SELL signal:                                                                                                                                                                                                                  
if self._is_position_open:       # only sell if we actually hold BTC                                                                                                                                                               
  self._is_position_open = False                                                                                                                                                                                                 
  → emit SELL order                                                                                                                                                                                                              
                                                                                                                                                                                                                                 
So the sequence is always:                                                                                                                                                                                                         
BUY → SELL → BUY → SELL → BUY → SELL ...
